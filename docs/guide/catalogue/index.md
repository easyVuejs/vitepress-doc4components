## 目录结构

<br>

```
.
├── src
│   ├── components (组件存放位置)
│   │   └── GeelySearch (查询组件)
│   │   │   ├── src (组件vue文件存放位置)
│   │   │   │   ├── GeelySearch.vue
│   │   │   │   └── GeelySearchGroup.vue
│   │   │   └── index.ts (单组件安装和输出)
│   │   └── index.ts (所有组件安装和输出)
│   ├── scripts (构建lib包，页面生成工具文件存放位置)
│   ├── views (可选，存放组件demo，方便开发环境调试)
│   │   └── GeelySearch
└── package.json

```
